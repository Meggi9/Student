# Пробный университетский IT-проект "Личный кабинет студента"

### Список технологий
* Java 
* Spring Boot
* Spring MVC
* Spring Security
* JPA (также JpaRepository)
* JDBC Template
* RowMapper
* Thymeleaf

### Функционал

* Авторизация/регистрация студентов
* Вывод личной информации о студенте
* Электронная зачётка
